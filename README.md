# Psychological Profile Modeling via OOP (Python)

This repository uses Object-Oriented Programming (OOP) to simulate and analyze complex psychological profiles, behavioral patterns, and personality disorders for educational and research purposes.

## 🚀 Available Profiles

1. **Narcissistic Personality Profile (`models/narcissism.py`)**
   - Models traits like ego inflation, lack of empathy, and the pursuit of "narcissistic supply".
   - Simulates core behavioral cycles: Love Bombing, Devaluation, Discard, and Narcissistic Rage (DARVO).

## 🛠️ How to Run the Simulation

Ensure you have Python installed, then run the simulation files:

```bash
python models/narcissism.py
